各位坛友好，从今早开始将不再发送散分贴了，将提供关于JSPGen软件开发框架的系列讲解内容。（回复者依旧有分.）

时间：二零一零年六月十三日 (早七点多) 天气：晴


一、知识扫盲：什么是JSPGenSDF？
  JSPGenSDF（JSPGen Software Development Framework）软件开发框架【简称JSPGen】，是一套独立的 Java(JSP) 软件开发框架，采用Java面向对象式开发技术，融入模型-视图-控制器 (MVC) 设计模式为敏捷性开发提供助力。框架以灵活、高效、安全性为目的快速开发方便、快捷、高效的应用软件。

二、当前最新版本为第三版，其中包含的核心有哪些？
  第三版框架由核心类、工具类和内置模块组成。提供了良好的可扩展性，同时想要使用别的类直接加入到应用包里面，就可直接通过框架配置文件来初始化后使用。

  第三版框架融入了新一代管理软件关注的重点思想，所有功能模块应用将与内置模块（管理体系、权限体系、模板引擎体系、表示逻辑体系、管理控制逻辑体系、扩展及个性化接口体系）充分结合，从框架的设计上优化软件系统实施成本，并通过内置模块，可实施个性化的软件构造，通过强大的日志记录、跟踪、可分析软件编写过程中遇到的各种问题，持续改善软件功能。

  目前第三版框架已成功应用到两个大型项目中，同时还在不断的完善框架本身，以适应于网络发展的需要。




  今早到此结束，明早继续！顺祝大家身体健康，万事如意，工作顺利，端午节愉快...
